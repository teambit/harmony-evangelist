### Overview

User testimonial. Showing a quote, user avatar, and name.  
Example:
```js live

	active
	data={{
		name: 'Barbra',
		description: 'head of barbering',
		content: 'Great scissors!',
		avatar: 'https://static.bit.dev/bit-logo.png',
	}}
/>
```
