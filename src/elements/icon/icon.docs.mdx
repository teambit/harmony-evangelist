### Overview

Placeholder for an icon, from Bit.dev's icon font.  
Make sure [IconFont](https://bit.dev/teambit/evangelist/theme/icon-font) is present in the page, for the icons to load.  
Spinner icon example:  
```js live
<ThemeCompositions>
	<Icon of="spinner" />
</ThemeCompositions>
```
