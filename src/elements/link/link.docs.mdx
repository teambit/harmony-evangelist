### Overview

A concrete link, styled for Evangelist, extending [base link](https://bit.cloud/teambit/base-ui/elements/link).  
Accepts all props as html Anchor Element.  

The link will use these css variables, when available:  
`--bit-accent-color`, for text color.  
`--bit-accent-heavy`, for text color on hover.  

Link with external prop example:
```js live
<Link href="https://google.com" external>look it up!</Link>
```
